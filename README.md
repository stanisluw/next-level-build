## О проекте

**NextLevelBuild** — это полнофункциональное веб-приложение для управления проектами и ресурсами сотрудников. Система позволяет руководителям и HR-специалистам отслеживать занятость команды, анализировать плановые и фактические показатели, а также принимать обоснованные решения о распределении ресурсов.

### Ключевые возможности

| Модуль | Описание |
|--------|----------|
|  **Проекты** | Таблица проектов с поиском, сортировкой и цветными статусами |
|  **Сотрудники** | Список сотрудников с должностями, стажем и переходом к статистике |
|  **Ресурсный учёт** | FTE расчёты, графики загрузки, детализация по неделям |
|  **Статистика сотрудника** | Линейные, столбчатые и круговые диаграммы (план/факт, выполнение задач) |
|  **Аутентификация** | Авторизация через localStorage, генерация аватарок |
